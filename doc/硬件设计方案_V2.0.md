三级断电架构（固态版）：
  · L0：米家 220V 智能插座（定时通断 12V 适配器）
  · L1：PUMP_EN MOSFET（PB1）= 水泵 12V 通路开关，替代原 CH1 继电器
  · L2：Z1~Z5 阀 MOSFET（单路 PWM 阶梯期间仅一路导通）

────────────────────────────────────────

3. 三级断电架构 (Three-Layer Cutoff)

| 级 | 名称 | 控制方 | 失效安全态 |
|---|---|---|---|
| **L0** | 米家 220V 智能插座 | 用户/米家APP | 开路（无市电） |
| **L1** | **PUMP_EN** MOSFET（PB1，AO3400） | MCU GPIO Low=OFF | GPIO 复位=Low->MOSFET截止->水泵无12V |
| **L2** | **Z1~Z5** 阀 MOSFET（AO3400） | MCU GPIO Low=OFF | GPIO 复位=Low->MOSFET截止->阀关闭（常闭阀自然关水）|

互锁铁律（软件强制）：
  1. PUMP_EN 使能前须至少一路阀（Z1~Z5）已开；
  2. 最后一路阀关闭前须先关 PUMP_EN；
  3. 复位/上电：全部 GPIO 默认 Low -> 失效安全 ✓。

6. 子模块详细接线 (Module Wiring)

6.1 TLP281-4 + AO3400 阀门/泵使能驱动电路

**器件选型**：

| 器件 | 型号 | 封装 | 关键参数 |
|---|---|---|---|
| 光耦 | TLP281-4 | SOP-16 | 4 通道独立；LED IF_max=50mA；NPN 输出 Vce_max=35V |
| 功率 MOS | AO3400 | SOT-23 | Vds=30V；Id=5.7A；Rds_on=46mΩ @Vgs=4.5V；Vgs(th)~1~2.5V |
| NPN 中间级 | S8050 | SOT-23 | Vceo=25V；Ic=1.5A；HFE≥40（反相驱动 MOSFET 栅极） |

**每路驱动电路（以阀 Z1/PB12 为例，泵使能路同理）：**

```
MCU PB12 --[R1=220Ω]-- TLP281_A(LED+)
                        TLP281_K(LED-) -- GND_ISO     (MCU 侧 GND_ISO)

TLP281 输出侧（GND_DIRTY 侧）：
  12V_DIRTY --[R2=47kΩ]-- TLP281_C --[R3=10kΩ]-- S8050_Base
  TLP281_E ─── GND_DIRTY
  S8050_Base --[R4=100kΩ]-- GND_DIRTY  (下拉防悬空)

  12V_DIRTY --[R5=10kΩ]-- S8050_C --[Rg=100Ω]-- AO3400_Gate
  AO3400_Gate --[R6=100kΩ]-- GND_DIRTY  (栅极下拉，复位态 MOSFET 截止)
  AO3400_Gate -- BZX84C15(15V Zener) -- GND_DIRTY  (Vgs 钳位保护)

  AO3400_Drain -- 阀线圈(+) -- 12V_DIRTY
  AO3400_Source -- GND_DIRTY
  1N5822 续流二极管：阴极接阀(+)，阳极接阀(-)，贴近阀本体安装
```

**逻辑真值表（失效安全验证）：**

| MCU GPIO | TLP281 | S8050 | AO3400_Gate | MOSFET | 阀（常闭型） | 安全？ |
|---|---|---|---|---|---|---|
| 0（复位默认） | OFF | ON（TLP281_C高→S8050导通→C极低） | ~0V | 截止 | 关（无12V）| ✅ 失效安全 |
| 1（开阀指令） | ON（TLP281_C低→S8050截止→C极高） | OFF | ~12V | 导通 | 开（加12V）| ✅ 正常 |

**关键：S8050 是反相器**，将 TLP281 输出的"高=截止"逻辑反转为 MCU 的"高=通"逻辑，
        同时恢复失效安全（GPIO=0 时 MOSFET 截止）。

6.2 TLP281-4 通道与 MCU 引脚映射

| IC | 通道 | MCU 引脚 | 排针位置 | 连接负载 |
|---|---|---|---|---|
| **IC1**（TLP281-4） | ch1 | PB12 | 左侧-1 | S8050 + AO3400 -> 阀 Z1 |
| IC1 | ch2 | PB13 | 左侧-2 | S8050 + AO3400 -> 阀 Z2 |
| IC1 | ch3 | PB14 | 左侧-3 | S8050 + AO3400 -> 阀 Z3 |
| IC1 | ch4 | PB15 | 左侧-4 | S8050 + AO3400 -> 阀 Z4 |
| **IC2**（TLP281-4） | ch1 | PA8 | 左侧-5 | S8050 + AO3400 -> 阀 Z5（备用） |
| IC2 | ch2 | PB1 | 右侧-35 | S8050 + AO3400 -> **PUMP_EN**（水泵12V使能） |
| IC2 | ch3 | PB0 | 右侧-34 | S8050 + AO3400 -> 备用（无负载） |
| IC2 | ch4 | -- | -- | 空置，备用 |

> **IC1 所有 4 路输入均来自左侧排针 PB12~PB15（连续）**，洞洞板 IC1 摆放在左侧即可完成最短连线。

6.3 PUMP_EN MOSFET（水泵 12V 总电源，PB1）

驱动电路与 6.1 阀门驱动完全相同，负载侧改为：
```
  AO3400_Drain ── LR7843 模块 DC+ 输入
  AO3400_Source ── GND_DIRTY
  SS54 续流二极管（阴极->DC+，阳极->GND）
  SMBJ24A TVS 跨接 LR7843 MOS 漏极（二级防护）
```

互锁：软件层（`Bsp_Valve_Set`）强制要求 PUMP_EN 使能前至少一路阀已开。

7. 供电分配网络 V2.0 (PDN)

**相比 V1.0 最大简化：去除 LM2596 5V_DIRTY 模块（无继电器线圈消耗）。**

```
12V/4A 适配器 (经 L0 米家插座)
    │
    ├─ NTC 5D-9
    ├─ 1812L300 自恢复保险
    ├─ 470µF/25V + 1µF 陶瓷
    ├─ SMBJ15CA TVS
    │
    └─> 12V_DIRTY 总母线
            │
            ├─> TLP281 输出侧偏置电阻（R2/R5 直接从 12V 取小电流，<0.5mA/路）
            │   各路 AO3400 负载侧（Drain 接阀线圈或 LR7843 DC+，Source 接 GND_DIRTY）
            │
            └─> B1205S-2WR3 隔离模块 IN+ -> 5V_ISO (200mA)
                    └─> 核心板 5V -> AMS1117 -> 3.3V (GND_ISO)
                                     MCU / OLED / EEPROM / TLP281 LED 侧
```

**保留 B1205S**：MCU 侧 GND_ISO 仍与 GND_DIRTY 隔离，TLP281 内部光学隔离维持双地架构。
